# SSL Checker Telegram Bot

Telegram-бот на Go выполняет разовую и периодическую проверку TLS/SSL сертификатов сайтов.
Он подключается к указанному URL или hostname, проверяет доверенную цепочку системным
хранилищем корневых сертификатов и возвращает сведения о leaf-сертификате, SAN,
issuer, сроках действия, fingerprint, протоколе TLS, cipher suite и цепочке.

## Возможности

- `/check <url|host[:port]>` — разовая проверка сертификата.
- `/watch <url|host[:port]> [минуты]` — периодическая проверка по таймеру.
- `/list` — список активных таймеров в текущем чате.
- `/unwatch <url|host[:port]>` — удаление таймера.
- Предупреждения об истечении, отсутствии SAN, несовпадении hostname и неполной цепочке.
- Нет внешних Go-зависимостей: бот использует стандартную библиотеку и Telegram Bot API через long polling.

## Запуск

```bash
export TELEGRAM_BOT_TOKEN="<token from BotFather>"
./go-ssl-check
```

Скрипт `go-ssl-check` сам соберет бинарник в `.bin/go-ssl-checker-bot` и запустит бота.
Для ручной проверки перед запуском можно выполнить:

```bash
go test ./...
go build -o /tmp/ssl-checker-bot .
```

По умолчанию `/watch` запускает проверку раз в 24 часа. Минимальный интервал — 1 минута.
